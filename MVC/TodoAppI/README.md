# TodoApp

This is a basic Todo application built with ASP.NET Core and a JavaScript frontend.

## Features
- Create, Read, Update, and Delete Todo items.
- In-memory database for quick setup.
- Minimal API endpoints.

## How to Run
1. Navigate to the `TodoAppI` project directory.
2. Run `dotnet run`.
3. Open your browser to the URL provided in the terminal (usually `http://localhost:5000` or `https://localhost:5001`).
